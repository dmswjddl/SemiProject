주제 : 전자기기 통합플랫폼

Problem : 전자기기 통합 플랫폼의 필요성	 

Customer Segment : COVID-19로 인한 재택근무자,
		비대면 수업을 받는 학생,
		전자기기 입문자,
		신규 전자기기 브랜드

Unique Value Proposition : 검색필터를 통해 고객이 원하는 제품 제공, 웹쇼핑의 편리성, 광범위한 전자기기 정보 제공

Solution : 다제품군의 효율적인 가격비교, 다양한 검색옵션 제공, 신규 전자기기 브랜드 입점용이

channels : youtube, facebook, twitter, 광고, 파워링크

Revenue Stream : 판매수수료, 신규브랜드입점 계약금, 광고료

Cost Structure : 서버유지비, 웹 개발 비용, 온/오프라인 마케팅 비용, 배송비용

Key Metrics : 방문자수, 구매자수, 방문자수 대비 실 구매자 비율, 온라인 설문조사

Unfair Advantage : 신규 전자기기 브랜드 접근성 용이, 다양한 검색 필터를 통한 구체적인 상품 비교, 사용자 위치 기반 희망브랜드 서비스센터/오프라인매장 노출
